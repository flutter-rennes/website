BreizhCamp

Dans ce cadre, l’association organise son événement annuel éponyme que vous connaissez sans doute vu que vous êtes sur le site web. Mais BreizhCamp c’est aussi un ensemble de groupes d’utilisateurs locaux qui organisent chaque mois des soirées techniques sur divers thèmes.
